# Syntecxhub Sales Performance Dashboard 📊

An interactive Data Science project that analyzes and visualizes retail sales data to uncover trends and business insights.

## 🚀 Features
* **Data Generation:** Custom script to create realistic retail datasets.
* **Interactive Dashboard:** Built with Streamlit for real-time data exploration.
* **Key Metrics:** Sales trends, product performance, and regional analysis.
* ## 💼 Project Overview

This project was developed as part of my Data Science internship task at Syntecxhub to analyze sales performance metrics and deliver actionable business insights.

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Pandas, Streamlit, Plotly
* **Tools:** GitHub, VS Code
